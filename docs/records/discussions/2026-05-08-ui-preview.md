# TallyClaw 宠物 UI 首版预览

日期：2026-05-08  
状态：预览提案

## 背景

项目所有者希望先看到 TallyClaw 宠物 UI 的视觉方向，再进入实际 UI 实现。

## 预览资产

- SVG 源文件：`assets/previews/tallyclaw-ui-preview-2026-05-08.svg`
- PNG 预览图：`assets/previews/tallyclaw-ui-preview-2026-05-08.png`

## 设计方向

首版预览采用“可拖动桌面宠物 + 悬浮观测胶囊”的组合：

- 中央是像素风 TallyClaw 宠物，保持蓝色科技感，但避免直接复制 Codex app 宠物形象。
- 左侧展示宠物状态与只读、低占用标签。
- 右侧展示今日 token、输入/输出、来源占比和准实时同步状态。
- 下方胶囊展示本周、请求、成功率、平均延迟四个核心指标。
- 详细 dashboard 暂缓，第一版优先保持桌面宠物低打扰。

## 后续待确认

- 宠物主体是否继续采用蓝色像素机器人方向。
- 悬浮统计胶囊是否默认常驻，还是仅在 hover/click 时展开。
- 第一版最小 UI 是否只保留宠物和一行状态，统计胶囊作为展开态。
